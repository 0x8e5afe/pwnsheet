# Notes Skeleton (OSCP)

Reflects the `buildNotesSkeleton('oscp')` branch in `script.js` and the "OSCP" option in the Notes Skeleton menu. Provides structured templates for lab-style standalone hosts and a small AD lab.

## Files created
```
pwnsheet-skeleton/
|- README.md
|- standalone/
|  |- standalone-01.md
|  |- standalone-02.md
|  |- standalone-03.md
|- AD/
|  |- general.md
|  |- credentials.md
|  |- machines/
|     |- ad-01.md
|     |- ad-02.md
|     |- ad-03.md
```

All templates are blank but contain the same headings, tables, and code sections used to collect OSCP-style enumeration, exploitation, and loot notes.
